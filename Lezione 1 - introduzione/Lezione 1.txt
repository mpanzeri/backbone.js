Cosa e' BackBone.js
------------------
BackBone.js e' un framework che ti permette di strutturare il tuo codice JavaScript secondo il pattern MVC.

Dipendenze richieste
--------------------
jquery
backbone
underscore

Installazione/configurazione
----------------------------

Come vedrai dal file index.html, bisogna importare i 3 file js dei 3 framework sopra citati.

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
    </head>
        <body>
            <script src="js/underscore.js"></script>
            <script src="js/jquery.js"> </script>
            <script src="js/backbone.js"></script>
        </body>
</html>

Verificare se l'installazione e' risucita
----------------------------------------

Andare su Chrome e sulla console javascript, digirare "Backbone" e se hai fatto tutto correttamente potrai avere accesso alle variabili globali di BackBone
